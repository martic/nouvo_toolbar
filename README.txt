README.txt
==========

An administrative toolbar built for use with Nouvo Theme.

INSTALLATION
============

 - Install like any other module: http://drupal.org/node/895232
 - Disable other "toolbar" modules

CONFIGURATION
=============

 - Configure user permissions
   - Access nouvo toolbar
     Users with this permission will see the nouvo toolbar.

ATTRIBUTION
===========

 - Toolbar module for most of the core module logic.
 - Icons by flaticon - http://www.flaticon.com/
